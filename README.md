# bluetoothGuys
## Кейс 1
## Bluetooth-мессенджер

Мобильное приложение для обмена текстовыми сообщениями по Bluetooth без подключения к интернету. Реализовать поиск устройств поблизости, установление соединения, отправку/получение сообщений, историю переписки в локальной БД.
